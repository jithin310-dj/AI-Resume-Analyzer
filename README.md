# 🚀 AI Resume Analyzer

AI-powered Resume Analyzer that uses NLP, semantic matching, and ATS scoring to evaluate resumes against job descriptions and provide actionable improvement suggestions.

## 🌐 Live Demo
https://ai-resume-analyzer-gui3boxdtnrk6gsfdx3k3k.streamlit.app/

## ✨ Features
- Resume Parsing (PDF)
- ATS Score Analysis
- Job Description Matching
- Keyword Gap Detection
- Semantic Similarity Analysis
- Personalized Resume Recommendations

## 🛠️ Tech Stack
- Python
- Streamlit
- NLP
- Scikit-learn
- Pandas
- PyPDF2
